SECURE ENTERPRISE MESSAGING APPLICATION

ROLE: LEAD DESIGNER

ABOUT THE PROJECT

“Secure Enterprise Messaging Application” is undertaken as a step towards the remedy of the security issues faced by any enterprise pertaining to misuse and loss of confidential data of the company to exterior parties. As a solution to this very naïve problem, we developed a cross platform mobile instant messaging application with enhanced focus on security. As a major feature of the application, only authorized users will be permitted to use the application. Also, a major security flaw of the current in market messaging applications i.e. all data (files, chat conversations, multimedia) can be stored and distributed locally, is addressed in our application. Thus, instead on reading from the local storage, everything will be read from the backend.


Tools and Technologies: Meteor, Ionic Framework, HTML, CSS, Angular JS
